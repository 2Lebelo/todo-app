# todo-app

## Overview
The Todo App is a simple and intuitive application that allows users to manage their daily tasks efficiently. Users can create a list of tasks they want to accomplish and mark them as completed once done. This app helps in staying organized and keeping track of important activities.

## Features
- **Add Tasks**: Easily create new tasks to include in your to-do list.  
- **Mark as Done**: Mark tasks as completed to track progress.  
- **View Tasks**: See all your tasks in a clean and organized list.  
- **User-Friendly Interface**: Designed for simplicity and ease of use.

## Usage
1. Open the app.  
2. Add a task by entering its description and clicking the “Add” button.  
3. Complete a task by marking it as done.  
4. Review your list to keep track of pending and completed tasks.
